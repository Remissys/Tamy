##  Requisitos

Node.js v22

## Frontend

Entrar no diretorio do frontend
```
cd ./frontend/
```

Baixar dependencias
```
npm ci
```

Rodar frontend
```
npm run dev
```

## Backend

Entrar no diretorio do backend
```
cd ./backend/
```

Baixar dependencias
```
npm ci
```

Rodar backend
```
npm run start
```
